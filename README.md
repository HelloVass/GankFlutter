# GankFlutter

[干货集中营](https://github.com/fujianlian/GankFlutter)是一款根据 Gank.io 官方提供的api实现的Gank客户端，包含最新数据展示，分类列表读取(Android，iOS，前端，休息视频，拓展资源，瞎推荐，App)，妹纸瀑布流图片功能。

后续版本会陆续实现历史干货，提交干货，github登录功能。
